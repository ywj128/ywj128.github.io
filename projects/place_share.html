Place Share Platform

I have developed a dynamic places sharing platform called Place Share using the MERN stack. This platform empowers users to share their favorite places with others, integrating AWS S3 for persistent image storage. Feel free to explore the website: Place Share.

Techonologies:

  • Frontend
    • React
    • Deployed on Render
  • Backend
    • Node.js
    • Express.js
    • MongoDB
    • AWS S3
    • Deployed on Firebase

Features:

  • Intuitive exploration of places posted by others.

  • Interactive view of places on Google Maps for enhanced user experience.

  • Sharing, editing, and deletion of places for users.

  • User authentication with signup and login capabilities.
